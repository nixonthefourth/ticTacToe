# Tic-Tac-Toe

This project shows the development of a simple turn-based tic-tac-toe game on vanilla Python 3.
Thinking behind the game is shown in the pseudocode and later implementation in main.py file.

# Project Structure
```
| main.py
| pseudocode.txt
| README.md
```

# Running
Make sure you install Python 3, preferrably 3.12.x.
Run `main.py` file.
